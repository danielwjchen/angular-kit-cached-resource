# Angular Kit Cached Resource
Angular Kit Cached Resource is a angular module that provides a function to create Cached resource.

## Development
To create a environment from scratch, install all the following dependencies.
### NVM and NPM
```
nvm install 5
```
### NPM global packages
```
npm install -g grunt-cli@1.2.0 grunt@0.4.5 npm@3.8.6
```
